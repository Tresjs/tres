Sensor works pretty similar to [collisions](/api/rigid-body/collisions) but don't generate contacts: they only generate intersection events when one sensor collider and another collider start/stop touching.
```html
<RigidBody activeCollision sensor @intersection-enter="onIntersectionEnter"          @intersection-exit="onIntersectionExit">
  <TresMesh>
    <TresTorusGeometry />
    <TresMeshNormalMaterial />
  </TresMesh>
</RigidBody>
```

::prose-note
Note you need to set the `activeCollision` prop to use sensors.
::

## Events

:::field-group
  ::::field{name="intersection-enter" type="(payload: { source: SourceTarget, target: SourceTarget }) => void"}
  Triggered when a sensor collider starts intersecting another collider.
  ::::

  ::::field{name="intersection-exit" type="(payload: { source: SourceTarget, target: SourceTarget }) => void"}
  Triggered when a sensor collider stops intersecting another collider.
  ::::
:::

Tresjs Rapier is a Vue-first wrapper around Rapier.js, a fast and robust physics engine for real-time 3D applications. It provides a clean, declarative API for rigid bodies, colliders, and joints so you can add realistic physics to TresJS scenes with minimal setup.

::prose-note
This library is based on Rapier, which is a Rust library ported to JavaScript using WASM, Rapier.js.
::

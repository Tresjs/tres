---
title: Installation
description: How to install and set up TresJS Post-processing
---

Follow these steps to install `@tresjs/post-processing`.

## Install Dependencies

```bash
pnpm add @tresjs/post-processing
```

## Basic Usage

```vue
<script setup lang="ts">
import { EffectComposerPmndrs, BloomPmndrs } from '@tresjs/post-processing'
</script>

<template>
  <TresCanvas>
    <!-- Your 3D scene -->
    <EffectComposerPmndrs>
      <BloomPmndrs :intensity="1.5" />
    </EffectComposerPmndrs>
  </TresCanvas>
</template>
```

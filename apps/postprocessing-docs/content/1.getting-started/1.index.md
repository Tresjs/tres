Welcome to TresJS Post-processing! This library provides a collection of ready-made post-processing effects as Vue components for TresJS.

## Quick Start

Get up and running in just a few minutes.

### Prerequisites

Before you begin, make sure you have:

- A TresJS project set up
- Node.js 18+ installed
- pnpm package manager
